ToDo MC Client is an unofficial Python library for accessing MS ToDo API.

.. note::

   The library is on early stage of development and isn't production ready.
   Look into `issues on Github`_ page to see current roadmap.

   .. _issues on Github: https://github.com/kam193/todo-ms-client/issues

-----
Usage
-----

The library is available from PIP::

   pip install todo-ms-client

See :doc:`reference/index` for this library API reference
